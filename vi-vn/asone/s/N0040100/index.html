    The DIP-MRES301B is a DIP module equipped with an AMR sensor (MRES301B) manufactured by Murata Corporation. Since it is a DIP shape, you can start the circuit experiment immediately with a bullet board. AMR (Anisotropic Magneto Resistive effect; anisotropic magnetoresistance) is a phenomenon in which the electrical resistance changes depending on the strength of a magnetic field. An AMR sensor is a magnetic sensor that uses an AMR magnetoresistive element. It can detect rotation and displacement by attaching a magnet to the rotating shaft of a motor and making it close to the magnet.
