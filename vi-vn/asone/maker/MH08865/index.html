Thùng chứa Polypropylene (PP) lý tưởng để sử dụng từ -10 ° C đến +50 ° C với nhiều mô hình có thể sử dụng từ -20 ° C đến +60 ° C (tùy thuộc vào tải và sử dụng). Các thùng chứa polyethylene mật độ cao (HDPE) là lý tưởng để sử dụng từ -30 ° C đến +30 ° C với nhiều mô hình có thể phục vụ từ -40 ° C đến +40 ° C (tùy thuộc vào tải và sử dụng).
