Đặc trưng

  • Flow liquid and slurry to a constant volume.
  • There is no grand seal valve and there is no worry of valve leakage or seal leakage.
  • With the action of the automatic check valve, there is no need to worry about backflow.
  • The flip-top method (up/down opening/closing method) allows tube replacement to be done smoothly with one touch.
  • Up to two heads can be installed (standard includes one head).
