Đặc trưng

  • This multi-logger can measure thermocouple or DC voltage of up to 12CH (humidity measurement is only 1CH using +5 V and 12CH ports).
  • You can measure graph display and list display by connecting to a computer with a USB cable (applications downloaded from the specified URL must be installed on the computer).
  • The recorded data is recorded in CSV format on either the SD card or the PC.
  • Recording can be transferred to the computer via the USB cable (recording is disabled).
  • With USB power supply, it is possible to supply power from various USB ports/portable batteries as well as the included AC adapter.
  • Further evolution! SDHC is now available

Thông số kỹ thuật

  • Size (mm): 167 x 91 x 35.8
  • Specifications (Body)
  • Thermocouple measurement: B, R, S, N, K, E, J, T (choose one)
  • Measuring temperature range: -200 to + 1370 degrees C (when using a K thermocouple)
  • Temperature measurement accuracy: +/- 0.05% of rdg + 1.0 degrees C (K: -100 to + 1370 degrees C at 25 degrees C)
  • DC Voltage Measurement: 100, 500 mV, 1, 5, 10, 50 V
  • DC voltage measurement accuracy: +/- 0.1% of F.S.
  • Humidity reading: 0 ~ 100% RH
  • Humidity measurement accuracy: +/- 5% RH (TYP) (10 ~ 95% RH at 23 degrees C)
  • Input channels: Analog/12 CH, + 5 V-OUT/1 CH for humidity measurement
  • Input terminal shape: Screw type terminal M3/* M 3.5 (outer dimension 6.2 mm or less/inner diameter 3.2 mm or more)
  • Sampling period: 5, 10, 30, 60 sec, 5, 10, 30, 60 min
  • PC I/F: USB 2.0 (Full Speed) Mini B Type
  • Card slot: 1 slot, SDHC compatible, available memory: 64 GB to 32 GB (not SDXC)
  • Memory: PC storage via SD card application
  • Recording format: Save in one file from the start of CSV recording
  • Records: 5 million (for 2 GB)
  • Accessories: simplified instruction manual, AC adapter, USB cable approx. 1M (for power supply and communication)
  • Specifications (Applications)
  • Operating System: Windows ® 8/8. 1/10 (32/64 bit)
  • Functions: Real-time measurement on application, recording start/stop, data storage, device settings, reading and displaying SD card data
  • Display method: Graph display (with overlay function), list display, maximum/minimum/average/comparison value display function of measured data
  • *Thermoelectric equivalent connecting wire is not included in the body. Prepare according to your intended use.
