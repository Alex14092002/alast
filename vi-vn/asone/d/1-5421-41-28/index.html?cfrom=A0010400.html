Đặc trưng

  • The temperature control uses PID control by a microcomputer, which provides high accuracy and stable performance.

  • The PV/SV two-stage display is used for the temperature display, so you can check the measured value and the set value at a glance.

  • Appendix: FAT report/Calibration Certificate of used measuring instrument copied
