Đặc trưng

  • The double lock structure of the dial type and the handle with the key provides excellent security.
  • The attached fixing metal fittings are useful for fall prevention in case of earthquake, etc.
  • Reagent bottles and other materials can be stored more securely by using stop bars and chemical trays sold separately.
