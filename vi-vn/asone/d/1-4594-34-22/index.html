Đặc trưng

  • The compact body is packed with useful features.
  • Sophisticated temperature control algorithms and error-free platinum temperature sensors enable precise temperature control.
  • Easy-to-use jog dial input system is used for temperature and timer settings for quick and reliable settings.
  • Microcomputer type PID control is used for highly accurate temperature control.
  • It is equipped with a notification buzzer when the set temperature and time are reached.
  • It employs a double-lock mechanism consisting of two independent circuits.
  • Four types of timer status (ON, OFF, ON/OFF, SET) are standard.
  • Beautiful and easy-to-view display screen. Uses a negative backlit LCD with good visibility.
  • You can also set the key lock and display the accumulated operating time, timer and remaining program time.
