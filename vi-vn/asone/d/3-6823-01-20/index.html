Đặc trưng

  • The body weight is about 250 g, making it lightweight, and the size fits in the pocket. Operation buttons are kept to a minimum.
  • With an intuitive design, 12 channels of terminal block and +5 V output for humidity sensor (M3 terminal block * 1)
  • This multi-logger can measure thermocouple or DC voltage of up to 12 channels. (For humidity measurement, only 1 channel is measured using +5 V and 12 channel ports.).
  • Recording can be transferred to the computer via the USB cable (recording is disabled).
  • By connecting the camera to a computer with a USB cable, you can perform measurements in graph view and list view. (The application downloaded from the specified URL must be installed on the PC.).
  • The recorded data is recorded in CSV format on either the SD card (MMC format) or the PC.
  • With USB power supply, power can be supplied not only from the included AC adapter, but also from various USB ports, portable batteries, etc.
  • There are two types of memory functions. You can save CSV files to an SD card when using a single device or to a PC when monitoring graphs.
  • Intuitive operation using the four-way controller
  • Large LCD with backlight
  • Not only can the graph of each channel be combined, but also the difference between the reference value and the reference CH can be displayed.
  • The temperature distribution at each position can be determined at a glance.
  • Like the main body, this button arrangement is designed for intuitive operation.
  • You can measure the relative humidity by preparing an optional humidity sensor. (1CH)
